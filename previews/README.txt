결과물 스크린샷 폴더
====================

AIS Ship Tracker와 태양계 시뮬레이터는 웹 앱이라
보고서 안에서 라이브로 바로 실행됩니다 (스크린샷 불필요).

스포츠 하이라이트만 스크린샷이 필요합니다.
아래 파일명으로 이 폴더에 저장하면 자동 표시됩니다:

  sports-highlight.png  ← 스포츠 하이라이트 결과 화면

저장 방법
---------
1) 앱/결과 화면을 띄운다.
2) [Win] + [Shift] + [S] → 영역 드래그 캡처.
3) 이 폴더에 위 파일명 그대로 저장 (.png).
4) index.html 새로고침 → 그 자리에 표시됩니다.

※ 파일이 없으면 "스크린샷 대기 중" 안내로 표시됩니다 — 오류 아님.
