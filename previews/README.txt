결과물 스크린샷 폴더
====================

이 폴더에 아래 파일명 그대로 스크린샷을 저장하면,
대시보드 "결과물 미리보기" 칸에 자동으로 표시됩니다.

  ais-pelmage.png       ← AIS Ship Tracker (PEL·MAGE 버전) 실행 화면
  ais-kyowa.png         ← AIS Ship Tracker (KYOWA LINE 버전) 실행 화면
  sports-highlight.png  ← 스포츠 하이라이트 결과 화면

저장 방법
---------
1) 해당 앱을 실행한다.
2) 키보드 [Win] + [Shift] + [S] → 영역을 드래그해 캡처.
   (또는 캡처 도구 / PrtScn 사용)
3) 이 폴더에 위 파일명과 '똑같이' 저장 (확장자 .png).
4) 대시보드(index.html)를 새로고침하면 그 자리에 그림이 박힙니다.

※ 파일이 없으면 그 칸은 "스크린샷 대기 중" 안내로 표시됩니다 — 오류 아님.
※ 태양계 시뮬레이터는 웹앱이라 스크린샷 없이 페이지 안에서 바로 실행됩니다.
