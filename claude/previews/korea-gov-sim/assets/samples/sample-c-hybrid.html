Hybrid LOD. 먼 zoom = SVG anchor pin (가벼움, 통일 langauge), 가까운 zoom = Three.js 미니어처 (입체, 인터랙티브). descentT 0.55 임계로 자연 전환. 44 entity 모두 anchor만 그리면 GPU 부담 없고, 사용자가 zoom in 한 cluster만 3D로 부풀어 오름.
pros: GPU 부담 ↓ (먼 zoom = SVG sprite) · 줌 인 시 입체감 ↑ · "이미지 파일" + 인터랙티브 둘 다 만족 · LOD 정합
cons: 2 시각 시스템 유지 (anchor SVG + Three building) · 디자인 일관성 신경 써야 · 전환 transition 작업 필요
